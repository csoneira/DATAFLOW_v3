===========================================
      mingo_analysis_bot - Command Guide
===========================================

Quick Start:
  /start or /help - Show this guide.
  /taskpdf - Interactive latest task PDF (station + task).
  /term <password> - Enable terminal mode.
  /exit - Disable terminal mode.
  /restart - Restart this bot.

Simulation PDFs (short commands):
  /mesh - Param mesh summary PDF.
  /sim_time - Simulation execution time histogram PDF.
  /backpressure - Latest STEP_0 backpressure monitor PDF.

Noise-Control PDFs (short commands):
  /noise_meta - Offender percent PDF.
  /noise_rate - Offender rate PDF.
  /noise_eff1 - Task 1 cumulative efficiency PDF.
  /noise_eff2 - Task 2 cumulative efficiency PDF.
  /noise_eff3 - Task 3 cumulative efficiency PDF.
  /noise_plane - Plane-combination rate PDF.

Other Analysis PDFs (short commands):
  /exec_map - Definitive execution map PDF.
  /filter_meta - Filter metadata PDF.
  /trig_rate - Trigger-rate metadata PDF.
  /config_meta - Configuration z-position metadata PDF.
  /rates_meta - Rates metadata PDF.
  /exec_meta - Execution metadata PDF.
  /eff_meta - Efficiency metadata PDF.
  /eff_34 - Three-to-four efficiency metadata PDF.
  /sim_evo - Simulated data evolution PDF.

Maintenance:
  /clean_status - Run clean_dataflow.sh --compact (disk usage check).
  /clean_force - Run clean_dataflow.sh --force --compact.

Compatibility:
  Legacy long command names remain supported.
===========================================
