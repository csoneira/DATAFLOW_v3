# Internal notes (not deployed)

This directory contains informal notes, to‑do lists, and experimental
observations that are **not part of the public documentation site**.  The
material here may be rough, conversational, or otherwise unsuitable for
external readers and is intended solely for internal reference by the
miniTRASGO/CASTRO collaboration.

Current files:

- `todo.md` – list of outstanding tasks and rough ideas.
- `to-understand.md` – notes on puzzling detector behaviour and
  investigation plans.

The docs build process ignores this directory; it does not appear in the
navigation sidebar and its contents are **not** published to GitHub Pages.

Feel free to move additional draft material here when refining the public
site.
