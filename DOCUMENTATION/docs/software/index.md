# Software

## Three pillars

1. **Analysis (software)**: `MASTER` processes real + simulated inputs; `STATIONS` stores station outputs/state.
2. **Simulation (digital twin)**: `MINGO_DIGITAL_TWIN` provides stepwise detector/electronics simulation.
3. **Dictionary-based inference (reconstruction)**: bridge between simulated knowledge and measured data.
