## Task 4 robust vs three-to-four efficiency is now physically consistent
---

### Figure
![Task 4 robust vs three-to-four for MINGO01 at z = 0/100/200/400 mm](FIGURES/2026_04_27_MINGO01_task4_robust_vs_three_to_four_z_0_100_200_400.png)

### Observation
This updated `MINGO01` Task 4 comparison at `z = 0/100/200/400 mm` now makes sense.

Before this correction, the plot was misleading because the missing-efficiency definition in the robust method was bad. That produced a relation with the three-to-four efficiency that did not behave consistently, especially when streamer activity changed.

After revising the Task 4 robust-efficiency construction, the comparison is now much more coherent:

- the time series evolve in a way that is physically plausible;
- the robust efficiency and the `four_plane / (four_plane + three_plane)` efficiency can now be compared directly;
- the real-data cloud can be interpreted against the simulation trend instead of looking like a broken estimator.

### Interpretation
This figure is more than a comparison plot. It is a quality criterion by itself.

The important point is not that the two efficiencies are numerically identical. The important point is that, for real data, they should lie on the same trend already established by the simulated case. If a point falls clearly outside that trend, that point should be treated as bad.

So the reading is:

- points following the simulated robust-vs-three-to-four relation are acceptable;
- points displaced away from that relation indicate problematic files or problematic efficiency reconstruction;
- this gives a direct quality diagnostic using only the internal consistency of the two efficiency definitions.

### Why this matters
This is useful for the next streamer-study step.

If the robust and three-to-four efficiencies remain mutually consistent in real data, then departures associated with streamers can be studied on a cleaner base. If they do not remain on the simulated trend, the file should first be treated as a quality problem instead of being interpreted as physics.
