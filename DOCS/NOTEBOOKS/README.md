# Research Notebooks

This folder contains chronological project notes, not canonical operational specs.

## Usage
- Keep one file per day using `YYYY_MM_DD_notebook.md` template.
- Treat notebook entries as context logs; migrate stable conclusions into canonical docs under:
  - `DOCS/REPO_DOCS/`
  - `MINGO_DIGITAL_TWIN/DOCS/`
  - `MINGO_DICTIONARY_CREATION_AND_TEST/DOCS/`

## Files
- Template: [`YYYY_MM_DD_notebook.md`](YYYY_MM_DD_notebook.md)
- 2026 March logs: `2026/03_MARCH/`
- 2026 February logs: `2026/02_FEBRUARY/`
